 
# ShadowByte - Automated Fruit Sorting System 
 
## Overview 
ShadowByte is an automated fruit sorting system that uses Computer Vision and Machine Learning to detect, classify, and sort fruits based on color, size, and ripeness. 
 
## Features 
- Real-time fruit detection 
- Classification based on color, size, and ripeness 
- High-accuracy ML model 
- Easy-to-extend architecture 
 
## Technologies Used 
- Python 
- OpenCV 
- TensorFlow / Keras 
- Matplotlib
